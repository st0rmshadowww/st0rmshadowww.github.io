1.安装node.js

最好通过nvm来完成不同版本的兼容与管理

比如gulp v3.9使用的node版本号需要小于等于v12；

而gulp v4.0+使用的node版本号需要大于v12；

可以通过nvm命令实现node版本的切换

a.检查当前node的版本号（版本前带星号的就是当前使用的node版本）

所有的node版本会保留在nvm的文件夹中

b.寻找可以下载的node版本号
c.检查当前npm的版本号
d.使用nvm进行版本管理

2.安装hexo-cli

切换npm镜像源使用cnpm进行软件包的下载

3.hexo init

关键步骤：初始化hexo

a.报错WARN git clone failed. Copying data instead

网络连接问题，重新尝试或者挂梯子，或者把github改成gitee，或者可以尝试手动复制在通过cnpm install下载依赖

b.报错端口占用

执行hexo server -p 4321 修改成一个可用的端口即可，当前网站部署在http://localhost:4321 处。（临时改变）
